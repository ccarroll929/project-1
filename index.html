About:

This web application allows users to find fitness exercises for a specific muscle/muscle group and also calculate the total number of calories they consume each day. Simply select a muscle option from the Select Muscle Group drop-down menu and click Get Exercises. Then you can enter a meal in the Meal entry box and then click the Add Meal button to add it to the list and the number of calories will be displayed! Continue adding meal items and the total number of calories in all meal items will be calculated.
